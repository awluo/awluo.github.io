Cloud Cat

A two-player platformer game

Role

Individual Project

Tools

Python, PyGame module

Duration

Spring 2019



How to Play

For my fundamentals of programming term project, I created a multi-player platformer game featuring a cat traversing through a world of clouds and enemies in the sky. Using the arrow keys to move and space bar (or AWSD and ctrl) to shoot, the player must try to advance as far as possible without falling or losing all three lives.


Game features
  • Dark Clouds: drop rain and disappear after being jumped on
  • Bees: shoot bullets and move up and down
  • Power-ups: point boost with coins and gain extra life with hearts
  • Level Editor: customize cloud platform pattern

Process

I sketched storyboard concepts and created frames for the main cat character's different action states.
